Third conditional

We use when we want expressed hypotetical situations in the past, regrets, gratefulness or reproach.

Result if conditional
would have/ wouldn't have + Past participle had/hadn't + Past Participle

!DANGER!

Never will/would with if

Regrets

I wouldn't have pass the exam if I had studied

Gratefulness

I wouldn't have pass the exam if I hadn't helped me

Reproach

You would have pass the exam if you had studied more

Have to VS must to

Both expressed * obligation*. We use must to when we think anything is very important. It is a personal belief.

I must to study more

We use have to when we wont to do anything but that isn't a personal belief. Someone or something forces us.

I have to study more

We wont but We study in order to pass te examan.
